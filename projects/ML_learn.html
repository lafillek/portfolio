LLM Fine-Tuning

Проект по дообучению языковой модели CodeLlama на специализированном наборе данных. Включает подготовку датасета, настройку обучения и сохранение адаптированной модели для дальнейшего использования.

Задача

Нужно было адаптировать базовую языковую модель под конкретную предметную область, сократить требования к ресурсам во время обучения и подготовить удобную конфигурацию для повторного запуска.

Что реализовано

  • Загрузка и настройка модели CodeLlama
  • Подготовка датасета в chat-формате
  • Дообучение модели с помощью LoRA
  • Квантование для снижения потребления памяти
  • Сохранение адаптера и токенизатора

Стек

Python PyTorch Transformers LoRA CodeLlama

Результат

Получилась рабочая конфигурация для дообучения модели, которую можно запускать повторно на новых данных и использовать в проектах, связанных с генерацией и анализом кода.
