Web-приложение для обработки изображений

Web-система для загрузки, хранения и распределённой обработки изображений. Проект использует базу данных и агентную архитектуру, где несколько обработчиков параллельно подготавливают изображения для дальнейшего обучения модели.

Задача

Нужно было создать систему, которая принимает изображения, сохраняет их в базе, распределяет задачи между агентами и формирует подготовленный датасет для последующего обучения ML-модели.

Что реализовано

  • Web-интерфейс для загрузки изображений
  • Хранение данных и статусов обработки в базе данных
  • Распределение задач между агентами
  • Параллельная обработка изображений
  • Подготовка данных для обучения модели

Стек

Python FastAPI PostgreSQL Docker OpenCV ML Pipeline
