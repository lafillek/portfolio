Telegram Bot Platform

Платформа для автоматизации клиентских заявок через Telegram. Проект включает бота, backend API, хранение данных и базовую админ-логику.

Задача

Заказчику нужно было автоматизировать приём заявок, сократить ручную обработку сообщений и сделать процесс общения с клиентами более быстрым.

Что реализовано

  • Telegram-бот для пользователей
  • Backend API для обработки заявок
  • Хранение данных в базе
  • Уведомления администратору
  • Базовая структура для масштабирования проекта

Стек

Python FastAPI PostgreSQL Telegram API Docker

Результат

Заказчик получил рабочий инструмент, который принимает заявки, сохраняет данные и снижает количество ручной работы.
